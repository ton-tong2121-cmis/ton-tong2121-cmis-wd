10 Steps Guide

  1. Clone you repository
  2. Make another copy of your html template file
  3. Name it index.html (Or any other name, but by default the webpage will show the index file)
  4. Add relavent html code
  5. Add in the contents you want to display with in the body element
  6. Start up a development server using the code "python -m SimpleHTTPServer" in the terminal
  7. Make sure the webpage is displaying the correct contents
  8. Validate you code using this website here
  9. Upload it to the production server using the command "sftp username@cmis-hsteacher"
  10. Get in to the public_html and move the relavent files over
